# JavaScript Advanced Core Concepts — Learning Repo

This is a beginner-friendly, code-along repository for advanced JavaScript topics . Each topic contains short notes, runnable examples and exercises so you can learn by doing.

Topics covered:

1. Prototype Chain (inheritance model)
2. ES6+ Syntax (destructuring, spread, optional chaining)
3. Iterators & Generators
4. ES Modules vs CommonJS
5. Practice exercises

## How to use

- Run each file with Node.js
- Read comments first
- Uncomment examples one-by-one
- Try exercises in `07-exercises` folder


## Quick guide & project hierarchy

Follow these quick steps to use the repo:

1. Open a topic folder (for example `01_prototype-chain`).
2. Read the `notes.md` file for a concise explanation.
3. Run `code.js` with Node.js or open `events.html` for browser examples.
4. Try the exercises in the relevant folder; check solutions when needed.
